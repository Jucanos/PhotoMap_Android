# PhotoMap_Android

# 앱이름

포토맵

# 간단한 설명

대한민국의 도별로 사진과 스토리를 생성하고 공유할 수 있어요 !

# 자세한 설명

대한민국의 전국팔도와 제주도에 해당 지역의 지도 모양으로 지도를 등록 할 수 있고, 지역별로 스토리를 등록할 수 있습니다. 또 그룹에 친구들을 초대해서 같은 지도를 공유할 수 있고 서로 스토리를 올리고 볼 수 있습니다.
